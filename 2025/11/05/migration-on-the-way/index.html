关于本站点即将迁移至 Halo 的通知

发表于2025-11-05|更新于2025-11-05|changelog
|总字数:476|阅读时长:1分钟|浏览量:

很高兴大家一直喜欢这个博客网站。为提供更稳定、高效的阅读体验，加上本人技术和时间等原因，网站框架将择期从目前的 Hexo 迁移至 Halo.

本次迁移的主要原因

1. 更完善的功能与管理体验

Hexo 作为静态博客非常轻量，但在内容管理、互动功能等方面有所局限。Halo 提供了强大的后台管理界面，支持更便捷的内容编辑与多语言管理，这将让我能更专注于创作本身。

2. 更优的站点性能

Halo 作为一个现代化的动态博客系统，在保持快速响应的同时，提供了比 Hexo 更丰富的原生功能，能更好地满足未来功能扩展的需求。

为什么我不选择其他框架？

1. 技术栈与运维方针的变更

之前我在一篇 changelog 里提到过，英文站点选择的是 WordPress 方案，目前迁移计划正在有条不紊的进行之中。为避免技术栈与运维方式的重复，中文（包括日文）站希望尝试一种更简洁、轻量的新方案。

2. 关于 Astro

诚然，Astro 性能卓越，但需要投入较多前端开发精力。作为一位多数使用 C/C++ 的新人开发者来说，这显然超出了我的技术能力范围。我希望能将时间更多地用于内容创作，而非如同在 Hexo 那样复杂的站点构建与维护。

写在最后

迁移期间，本站访问可能会出现短暂中断，敬请谅解。全新的站点将保留绝大多数现有内容与链接（所有的 changelog 将置于一个归档链接内），访问体验将更为流畅。

再次感谢各位的理解与支持！
